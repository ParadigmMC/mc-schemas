# mc-schemas

This repository contains schemas for all sorts of Minecraft related formats and configuration files.

## Folder Structure

Basically: `{type}/[{owner}/{name}]/{side}/**/*.json`

- `minecraft` - Schemas of vanilla
  - `server`
    - `server.properties.json`
    - `eula.txt.json`
  - `client`
    - `options.txt.json`
